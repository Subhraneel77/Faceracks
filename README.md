# Faceracks

Running the Examples :-

Clone the repository :- https://github.com/Subhraneel77/Faceracks.git 

Running the Browser Examples :- 

1) cd Faceracks/examples/examples-browser
2) npm i
3) npm start

LOCAL HOST :- http://localhost:3000/
